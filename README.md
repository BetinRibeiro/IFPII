<h1>Teste de Personalidade | Projeto JS + Tailwind</h1>

<p>Este projeto é um quiz interativo de traços de personalidade, desenvolvido em HTML, CSS (Tailwind) e JavaScript puro.  
O usuário responde a uma série de afirmações e recebe uma média para cada categoria psicológica ao final.</p>

<h2>Funcionalidades</h2>

<ul>
 <li>Interface moderna com Tailwind</li>
 <li>Sistema de perguntas com pontuação 1 a 5</li>
 <li>Cálculo automático de médias por categoria</li>
 <li>Classificação se o candidato está "Apto" ou "Não Apto"</li>
 <li>Barra de progresso animada</li>
 <li>Download do resultado em arquivo .txt</li>
 <li>Suporte a várias dimensões de personalidade</li>
</ul>

<h2>Estrutura do Projeto</h2>

<pre>
index.html
standards.html
assets/
 └── script.js (arquivo do quiz)
</pre>

<h2>Como usar</h2>

<ol>
 <li>Abra o arquivo <strong>index.html</strong></li>
 <li>Clique em “Iniciar teste”</li>
 <li>Responda as perguntas clicando nas notas (1 a 5)</li>
 <li>No fim, veja o resultado e baixe o arquivo txt se quiser</li>
</ol>
